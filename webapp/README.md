## How to Run the WebApp:

Go to `webapp` folder.

### Frontend

  ```shell
  $ cd frontend
  $ npm install #you only need to run this the first time
  $ npm start
  ```
  
### Backend

Use a virtual environment. Note: You may need to use `python3` instead of `python`.

  ```shell
  $ cd backend
  $ python manage.py runserver
  ```
