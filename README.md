# christmas-lights
Programmable Christmas Lights.

## How to Run the WebApp:

Go to `webapp` folder.

### Frontend

  ```
  $ cd frontend
  $ npm start
  ```
  
### Backend

Use a virtual environment. Note: You may need to use `python3` instead of `python`.

  ```
  $ cd backend
  $ python manage.py runserver
  ```
